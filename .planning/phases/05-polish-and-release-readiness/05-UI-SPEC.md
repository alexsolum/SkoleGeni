## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline chemistry tag padding, dense list row gaps |
| sm | 8px | Compact element spacing, button icon-label gap, stat card internal padding |
| md | 16px | Default element spacing, constraint card padding, table cell padding |
| lg | 24px | Section padding, sidebar header padding, card group gaps |
| xl | 32px | Layout gaps between major page sections |
| 2xl | 48px | Major section breaks (e.g., between score cards and class breakdown table) |
| 3xl | 64px | Page-level spacing; action bar height |

Exceptions:
- Sidebar nav link height: 36px (py-2 = 8px top + 8px bottom on 20px line-height text)
- Button height (primary CTA): 44px — PRD explicitly specifies 44px for Welcome CTA touch target
- Metric card height: 140px — PRD spec for Results metric cards
- Project card height on Welcome: 64px — PRD spec

Source: PRD.md Design Tokens, 05-CONTEXT.md dense spacing directive.

## Typography

4 sizes, 2 weights. The `Button` row shares the Body size (14px) and is listed separately for usage clarity only — it does not constitute a fifth size.

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Display | 32px | 700 | 1.2 | Cabinet Grotesk | Welcome screen heading, page-level titles |
| Heading | 24px | 700 | 1.2 | Cabinet Grotesk | Section headings, card titles, class names |
| Body | 14px | 400 | 1.5 | Satoshi | Table content, form labels, descriptions, nav labels |
| Small | 12px | 400 | 1.5 | Satoshi | Helper text, muted subtitles, sidebar app subtitle |
| Button | 14px | 700 | — | Cabinet Grotesk | All button labels (matches existing Button.tsx pattern) — same size as Body |

Numeric data (scores, counts, capacity values, percentages) uses JetBrains Mono at 14px weight 400. The monospace font-family alone distinguishes numeric data from body text — no separate size tier is needed.

Source: PRD.md Typography section, 05-CONTEXT.md font decisions.

Note: "Satoshi" replaces "Manrope" in both `src/index.css` and `tailwind.config.js`. Sourced via Fontshare CDN. Cabinet Grotesk remains on Google Fonts unless weights are unavailable there (see 05-RESEARCH.md Open Question 1).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F8FAFC | Page background (`bg-background`), pupil card hover, class column background |
| Secondary (30%) | #FFFFFF | Cards (`surface`), sidebar background, table row background, modal backgrounds, metric cards |
| Accent (10%) | #2563EB | Reserved — see list below |
| Border | #E2E8F0 | All 1px solid borders — cards, sidebar divider, table rows, constraint cards |
| Destructive | #DC2626 | Error toasts, error state text, Red Card pupil highlight background (#FEE2E2 fill) |
| Text primary | #334155 | All body copy, table content |
| Text muted | #94A3B8 | Helper text, sidebar subtitle, disabled states |
| Primary dark | #0F172A | Primary buttons, sidebar app name, page headings |

Accent (#2563EB) reserved for:
- Active sidebar nav link (left border indicator + text + `bg-accent/10` fill)
- Focus rings on form inputs (1px accent border on focus)
- Slider track color (class size range slider)
- Run Optimizer CTA button background
- Sub-score progress bar fill
- Overall Score radial SVG stroke
- Text links within content areas
- Health bar fill for classes scoring >= 0.8 on the Results breakdown table

Second semantic color: #DC2626 (destructive) — used for error toasts only, Red Card states use background #FEE2E2 with #DC2626 text.

Health bar thresholds (Results class breakdown table):
- >= 0.8: `bg-green-500`
- >= 0.6: `bg-amber-500`
- < 0.6: `bg-red-500`

Source: PRD.md Color Palette, 05-CONTEXT.md decisions, 05-RESEARCH.md health bar thresholds.

Table row height: 48px. Header row: Satoshi 12px 400 #94A3B8 uppercase.

## Interaction Contract

### Sidebar Nav

- `NavLink` from react-router-dom — `isActive` prop drives active state styling
- Active style applied immediately on navigation (no animation)
- Sidebar is static — no collapse, no hover expand

### Chemistry Stat Cards

- Read-only display — no interactions. Derived from in-memory chemistry state already loaded by PupilData.tsx.

### Class Breakdown Table — View Classes Link

- Click: routes to `/editor/:projectId` for the corresponding class
- Hover: underline on "View Classes" text

### Health Progress Bar

- Static visual — width set by health score percentage
- Color determined by threshold: green >= 0.8, amber >= 0.6, red < 0.6
- No animation required

### Font Loading

- `font-display: swap` (Fontshare CDN default) — brief FOUT is acceptable for desktop-first internal tool
- Do not block render for font loading

## Anti-Patterns (Do Not)

- Do not animate the sidebar (no slide-in, no collapse)
- Do not apply AppShell to the Welcome route (`/`)
- Do not use `drop-shadow` utilities — use `border border-[#E2E8F0]` exclusively
- Do not add new font weights beyond what is imported (400, 700 for Satoshi; 400, 700 for Cabinet Grotesk; 400 for JetBrains Mono)
- Do not use color values outside the declared palette without documenting the exception
- Do not introduce a fifth font size tier — numeric data uses JetBrains Mono at existing declared sizes (14px body, 24px heading)
