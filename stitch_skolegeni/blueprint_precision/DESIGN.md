# Design System Document

This document outlines the core design principles and token values for our user interface.

## Color Palette

Our color palette is designed to be vibrant and accessible, ensuring a consistent visual experience across all platforms.

-   **Color Mode**: `light`
-   **Primary Color**: `#2463eb` - This is the brand's most distinctive chromatic color, suitable for buttons, CTAs, and key interactive elements.
-   **Secondary Color**: `#6275af` - A supporting color for less prominent UI elements, chips, and secondary actions.
-   **Tertiary Color**: `#bc4800` - An additional accent color for highlights, badges, or decorative elements.
-   **Neutral Color**: `#757681` - A neutral base color for backgrounds, surfaces, and non-chromatic elements.

## Typography

We use a carefully selected set of typefaces to maintain readability and brand consistency.

-   **Headline Font**: `inter` - Used for all major headings and titles.
-   **Body Font**: `inter` - Applied to standard paragraph text and general content.
-   **Label Font**: `inter` - Utilized for labels, small text, and UI elements.

## Shape and Spacing

The physical attributes of our UI components are defined by their roundedness and the spacing between elements.

-   **Roundedness**: `1` (Subtle roundedness) - Corners of UI shapes feature a subtle rounding, offering a modern yet approachable feel.
-   **Spacing**: `2` (Normal) - We maintain a normal level of whitespace within and between UI elements, striking a balance between density and readability.
